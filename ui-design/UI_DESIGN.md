## Key Design Principles

1. **Minimalist** - Clean, no clutter
2. **Orange Accent** - Mars/Ares theme color
3. **Subtle Interactions** - Multitask button doesn't look like a button
4. **Consistent** - Same orange throughout
5. **Beautiful** - Mars-themed wallpaper, elegant typography

---

## Panel Design Details

```
┌─┬────────────────────────────────────────────────────────────┐
│█│ Filer  Menu  Settings  Help              Tuesday,12 10:15 PM│
└─┴────────────────────────────────────────────────────────────┘
 ↑
 Orange multitask button (vertical bar, full height)
```

**NOT** a traditional button - just a clean orange accent that's clickable!

---

## Implementation Notes

### Multitask Button
- Full height vertical bar
- Orange color (#D4622A or similar)
- No border, no hover effect that makes it look like a button
- Just a clean accent that happens to be interactive

### System Tray
- NOT visible in mockups
- May be hidden or not implemented yet
- Focus on clean, minimal design first

### Colors
- Orange accent: `#D4622A` (Mars orange)
- Background: Dark/black for boot/shutdown
- Panel: Subtle dark background
- Text: White/light for contrast
